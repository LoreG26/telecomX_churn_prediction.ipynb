# Telecom X - Churn Analysis and Prediction

## Project Description

This project analyzes customer churn in the Telecom X company. The objective is to understand the main factors that influence customer cancellation and build predictive models capable of identifying clients with a high probability of leaving the service.

The project was developed as part of a Data Science challenge focusing on ETL processes, exploratory data analysis and machine learning modeling.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Project Structure

Part 1: Data Cleaning and Exploratory Analysis  
- Data extraction from JSON API  
- Data cleaning and transformation  
- Exploratory Data Analysis (EDA)  
- Data visualization  

Part 2: Machine Learning Models  
- Data preprocessing  
- Feature encoding  
- Data normalization  
- Training predictive models  
- Model evaluation using metrics  

## Machine Learning Models

Two models were trained to predict customer churn:

- Logistic Regression
- Random Forest Classifier

These models were evaluated using:

- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

## Key Insights

The analysis suggests that several factors influence customer churn, including:

- Contract type
- Monthly charges
- Customer tenure
- Service configuration

Understanding these factors can help the company design retention strategies and reduce customer churn.
